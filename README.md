# Legacy Windows Exposure Assessment Tool

## Purpose
The tool performs non-intrusive checks for commonly exposed legacy services such as SMB, NetBIOS, and RDP. It maps detected services to known security risks and generates structured reports.

## Features
- Exposure-based assessment (no exploitation)
- Severity-based risk scoring
- HTML, CSV, and JSON report generation
- Designed for legacy Windows environments

## Ethical Note
This tool is intended strictly for educational and authorized assessment purposes.
